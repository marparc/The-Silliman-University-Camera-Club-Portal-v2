## The-Silliman-University-Camera-Club-Portal-v2

A full-stack, AI-enhanced management and automation platform for the Silliman University Camera Club that streamlines bookings, event coordination, member management, and analytics.

The system improves shooter assignment and eligibility matching using rule-based logic and AI-assisted recommendations based on skills and availability. It also features an intelligent equipment management module that suggests compatible gear based on event requirements and user profiles.

V2 integrates AI automation with Google Calendar, Gmail, Facebook, Discord, and Google Meet to enable automated scheduling, notifications, and communication across platforms. It includes secure authentication, role-based access control, personalized user settings, a booking and deliverables tracking system, and an analytics dashboard for data-driven decision-making.

Built with a modern full-stack architecture, the platform transforms club operations into a more intelligent, automated, and efficient workflow system.
